# Triage 标签

各 skill 使用五个规范 triage 角色。本文件把这些角色映射到本仓库实际使用的标签字符串。

| skills 中的标签 | 本仓库标签 | 含义 |
| --- | --- | --- |
| `needs-triage` | `needs-triage` | 待维护者评估 |
| `needs-info` | `needs-info` | 等待报告者补充信息 |
| `ready-for-agent` | `ready-for-agent` | 已完整定义,可交给 AFK agent 实施 |
| `ready-for-human` | `ready-for-human` | 需要人类实施 |
| `wontfix` | `wontfix` | 不予处理 |

当 skill 提到某个角色(如"打上 AFK-ready triage 标签")时,使用表中对应的标签字符串。
