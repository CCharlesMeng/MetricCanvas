# 页面模板引用已发布页面修订而不复制页面文档

页面模板的模板修订保存发现元数据与精确的已发布页面修订引用,不复制页面文档。模板发布代表模板管理员批准在模板 ACL 范围内复用该结构;来源看板页面后续产生或发布新页面修订时,既有模板修订仍指向原来的不可变页面修订。

## Considered Options

- 在模板修订中复制页面文档:读取直接,但会产生第二份页面文档真源,后续无法判断模板内容与来源修订的关系,被否决。
- 始终引用来源看板页面的当前发布修订:模板会随来源页面静默变化,破坏发布确认的精确性,被否决。

## Consequences

- 保存模板修订时必须验证目标页面修订曾成功发布。
- `search_templates` 按模板 ACL 返回精确来源页面修订,不要求调用者同时拥有来源看板页面权限。
- 从模板创建看板页面时仍需形成新的页面文档,并按当前元数据重新校验。
