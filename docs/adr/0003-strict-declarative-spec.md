# 页面规格采用严格声明式,禁止表达式与脚本

页面规格只表达查询引用、布局、图表类型和筛选联动;不允许内嵌表达式、脚本或自定义样式。同环比、衍生计算等复杂逻辑一律下沉到表服务的指标定义中。

这是刻意收窄:表达力换可控性。规格因此可被 JSON Schema 完整校验,AI 生成结果可控、可自动修复,DSL 不会膨胀为"用 JSON 写的编程语言"。代价是部分需求(如临时衍生指标)必须走表服务定义流程,不能在页面层"顺手写一个"。

## Considered Options

- 受限表达式(同环比、预警变色写在规格里):灵活但校验困难,且表达式能力一旦开放极难收回,被否决。
- 自定义组件逃生舱:与"页面不再以源代码为核心资产"的调整目标矛盾,被否决。
